# ABOUT #

Gulp template

## INSTALLATION

	git clone https://github.com/bogachenkov/gulp-template.git
	npm i


## USAGE

Run in developing mode ( `app/` folder)

	gulp

Production build ( `dist/` folder)

	gulp build
	
## PROJECT STRUCTURE

	
	app/
	--css/
	--fonts/
	--img/
	--js/
	--libs/
	--sass/
	dist/
